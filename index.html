Redox Donors Notebooks

The Redox Donors Notebooks repository contains jupyter notebooks coded in python, and data I collated for a review about sacrificial electron donors. Over the years, many of my photochemistry colleagues had asked for a list of redox potentials for these. This should hopefully be the list they wanted. At the time I was working in a data science team, so I couldn't resist playing with the data to make useful plots.

simple_uno_pump

I wrote this code in C++ when an arduino syringe pump broke and I had to replace a component. It was just after covid and because of delivery issues I had to replace it with a completely different type of component. This meant I had to alter the existing code. However, the original code was so beyond my basic C++ skills at the time that I ended up re-writing a more basic version. I put it on github in the hope that it might help someone else in a similar position in the future.
